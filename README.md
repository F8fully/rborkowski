# My CV / Resume

This repository contains my personal CV built using HTML, CSS and JavaScript.

The project is designed to:
- Present my professional experience and skills
- Serve as an online portfolio
- Generate a downloadable PDF version using html2pdf.js

---

## 🌐 Live Preview

You can view the online version here:

👉 [Live CV](https://f8fully.github.io/RBorkowski_CV/)

---

## 📄 Features

- Modern responsive design
- Editable content (contenteditable sections)
- Skill classification (Hard / Mid / Low / Soft skills)
- PDF download functionality
- Clean A4 layout optimized for printing

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- html2pdf.js

---

## 📥 Generate PDF

Click the **PDF** button on the website to generate and download the resume as an A4 PDF file.

---

## 📌 Purpose

This project is a personal resume and portfolio representation.
It is intended for presentation and job application purposes.
